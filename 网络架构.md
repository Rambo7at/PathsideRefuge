# 途畔归所 网络同步框架文档（v2 原生RPC版）

## 一、总体架构

| 模块/单例 | 类型 | 核心职责 | 关键原生RPC | 备注 |
| :--- | :--- | :--- | :--- | :--- |
| **NetCore** | Autoload `Node` | 连接管理；少数全局RPC入口 | `RequestSpawnPlayer` (AnyPeer) | 不管理对象，不处理同步数据 |
| **NetObjectRegistry** | Autoload `Node` | 对象注册表；生命周期广播；周期性状态同步（待激活） | `BroadcastCreate(byte[])` (AnyPeer)<br>`BroadcastDestroy(byte[])` (AnyPeer) | 对象同步的唯一权威 |
| **NetObjectManager** | Autoload `Node` | 预制体字典；监听OnSpawned/OnDestroyed；实例化/销毁节点 | 无 | 纯粹本地工厂 |
| **PlayerManager** | Autoload `Node` | 玩家生成封装 | 无 | 内部转调NetObjectRegistry |
| **NetSyncBase** | `Node3D` (组件) | 网络身份标签（NetID, IsOwner）；注册到Registry | 无 | **不再处理任何RPC收发** |
| **业务脚本** | 挂在预制体上的自定义类 | 定义`[Rpc]`方法，直接调用`Rpc()`/`RpcId()`实现对象通信 | 自定义 | 取代旧NetSyncBase.RPC系统 |
